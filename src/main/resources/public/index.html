The project is a Blog

  • The blog has an admin section and a public section
  • The admin can add blog posts and modify/delete them
  • The admin also has an overview of all posts
  • The normal user can see a list of all blog posts
  • The user can choose to read more about a blog post (all details)
  • The user can add comments to a blog post
  • The user (and admin) can see the "About Me" page
